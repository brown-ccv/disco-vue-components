# Stacked Bar Chart

A Stacked Bar Chart shows aggregated data over categorical values on the x axis.

<ArgsTable story="Stacked Bar Chart" />

## DStackedBarChart component

```html
<DStackedBarChart
  id="stackedbarchart"
  :dataset="[
    { category: 'A', amount: 28},
    { category: 'B', amount: 55},
    { category: 'C', amount: 43}
  ]"
  :min-width="400"
  x="category"
  x-label="Letters"
  y="amount"
  y-label="Amount"
  :include-actions="true"
/>
```
